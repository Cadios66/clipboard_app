#  Clipy

**Clipy** — менеджер буфера обмена для Windows. Программа автоматически индексирует всё, что вы копируете, и структурирует данные для быстрого доступа.

---

## Особенности

* Сохранение текста и изображений в локальную базу данных с автоматической сортировкой по датам.
* Автоматическое удаление записей для экономии места на диске
* Работа в фоновом режиме
* Возможность автоматического старта вместе с Windows через системный реестр.
* Фильтрация по тексу/датам, поиск по ключевым словам
* Кастомизация под любимый цвет

---

## Технический стек

| Библиотека | Назначение |
| :--- | :--- |
| **CustomTkinter** | Современный графический интерфейс с поддержкой тем |
| **Socket** | Межпроцессное взаимодействие и блокировка порта |
| **Threading** | Параллельный мониторинг буфера  |
| **WinReg** | Взаимодействие с реестром Windows для автозагрузки |
| **JSON/OS** | Работа с файловой системой и хранение конфигураций |

---



##  Установка и запуск

1. Скачайте последний релиз приложения
2. Запустить `Clipy_app.exe`
3. При первом запуске программа создаст все необходимые конфигурационные файлы автоматически
4. Выбрать папку, в которую буду сохранятся ваши данные
5. Открыть программу вновь можно из трея
