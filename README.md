#  Clipy

**Clipy** — менеджер буфера обмена для Windows. Программа автоматически сохраняет всё, что вы копируете, и структурирует данные для быстрого доступа.

---

## Особенности

* Сохранение текста и изображений в локальную базу данных с автоматической сортировкой по датам.
* Автоматическое удаление записей для экономии места на диске
* Работа в фоновом режиме
* Возможность автоматического старта вместе с Windows через системный реестр.
* Фильтрация по тексу/датам, поиск по ключевым словам
* Кастомизация под любимый цвет

---

## Технический стек
* Python 3.14

### Библиотеки
* Внешние: Pillow (PIL), CustomTkinter (ctk), PyWin32 (win32clipboard, winreg), Pystray

* Встроенные: tkinter, os, sys, shutil, datetime (date, timedelta), threading, hashlib, json, io, math, time, colorsys, socket 

---



##  Установка и запуск

1. Скачать последний релиз приложения
2. Запустить `Clipy.exe`
3. При первом запуске программа создаст все необходимые конфигурационные файлы автоматически
4. Выбрать папку, в которую будут сохранятся ваши данные
5. Открыть программу вновь можно из трея

Если вы скачиваете готовую версию в формате .exe, ваш антивирус может ругаться и помечать файл как «Trojan» или «Unsafe». Поскольку у программы нет платного сертификата разработчика, Windows не узнает издателя

* Что можно сделать?

1. Добавить файл в исключения антивируса.
2. Скачать исходный код (.py файлы), установить все библиотеки и запустить через Python
