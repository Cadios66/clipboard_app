#  Clipy — Smart Clipboard Manager

**Clipy** — это легковесный и надежный менеджер буфера обмена для Windows. Программа автоматически индексирует всё, что вы копируете, и структурирует данные для быстрого доступа, освобождая вас от страха потерять важную информацию.

---

## Ключевые особенности

* **Умное архивирование:** Сохранение текста и изображений в локальную базу данных с автоматической сортировкой по датам.
* **Авто-клининг:** Интеллектуальное удаление записей старше 7 дней для экономии места на диске (реализовано через `datetime`).
* **Single Instance Mode:** Защита от дублирования процессов с использованием **TCP-сокетов** (порт 65432).
* **Фоновый режим:** Удобная работа через системный трей (Tray Icon) — программа не мешает работе, но всегда под рукой.
* **Автозагрузка:** Возможность автоматического старта вместе с Windows через системный реестр.

---

## Технический стек

| Библиотека | Назначение |
| :--- | :--- |
| **CustomTkinter** | Современный графический интерфейс с поддержкой тем. |
| **Socket** | Межпроцессное взаимодействие (IPC) и блокировка порта. |
| **Threading** | Параллельный мониторинг буфера без фризов интерфейса. |
| **WinReg** | Прямое взаимодействие с реестром Windows для автозагрузки. |
| **JSON/OS** | Работа с файловой системой и хранение конфигураций. |

---



##  Установка и запуск

1. Скачайте последний релиз из папки `dist`.
2. Запустите `Clipy.exe`.
3. При первом запуске программа создаст все необходимые конфигурационные файлы автоматически.
4. Открыть программу вновь можно из трея
